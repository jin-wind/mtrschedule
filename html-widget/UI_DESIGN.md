## 顏色方案

### 背景
- **主背景**: 紫色漸變 (#667eea → #764ba2)
- **小組件容器**: 半透明白色 (rgba(255, 255, 255, 0.95)) + 毛玻璃效果

### 標題列
- **背景**: 紫色漸變 (#667eea → #764ba2)
- **按鈕**: 半透明白色邊框，懸停時放大

### 月台
- **月台 1**: 
  - 標題顏色: #ef4444 (紅色)
  - 卡片左邊框: #ef4444
  - 背景漸變: 淡紅色 (rgba(239, 68, 68, 0.03))

- **月台 2**:
  - 標題顏色: #3b82f6 (藍色)
  - 卡片左邊框: #3b82f6
  - 背景漸變: 淡藍色 (rgba(59, 130, 246, 0.03))

### 列車卡片
- **背景**: 白色
- **陰影**: 0 2px 8px rgba(0, 0, 0, 0.08)
- **懸停效果**: 向右移動 3px，陰影增強
- **動畫**: slideIn 0.4s ease (從下方滑入)

## 動畫效果

### 1. 卡片進入動畫
```css
@keyframes slideIn {
    from {
        opacity: 0;
        transform: translateY(10px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
```

### 2. 載入動畫
```css
@keyframes pulse {
    0%, 100% { opacity: 1; }
    50% { opacity: 0.5; }
}
```

### 3. 懸停效果
- 車站下拉選單: 向上移動 1px + 陰影
- 刷新/置頂按鈕: 放大 1.1 倍
- 列車卡片: 向右移動 3px + 陰影增強

## 字體大小

- **路線號碼**: 24px (粗體 900)
- **目的地**: 15px (粗體 600)
- **到站時間**: 13px (粗體 600，綠色)
- **車廂類型**: 13px
- **月台標題**: 16px (粗體 700)
- **更新時間**: 12px

## 響應式設計

### 桌面模式 (寬度 > 768px)
- 雙欄佈局
- 小組件寬度: 750px
- 月台並排顯示

### 移動模式 (寬度 ≤ 768px)
- 單欄佈局
- 寬度: 100%
- 月台上下排列

## 特殊效果

### 毛玻璃效果
```css
backdrop-filter: blur(10px);
```

### 圓角
- 小組件容器: 16px
- 車站下拉選單: 8px
- 按鈕: 8px
- 列車卡片: 10px

### 陰影
- 小組件容器: 0 8px 32px rgba(0, 0, 0, 0.2)
- 列車卡片: 0 2px 8px rgba(0, 0, 0, 0.08)
- 懸停時卡片: 0 4px 12px rgba(0, 0, 0, 0.12)

## 互動元素

### 按鈕
- **刷新按鈕** (🔄): 手動刷新時刻表
- **置頂按鈕** (📌): 固定視窗（在桌面應用中有效）

### 車站選擇
- 下拉選單包含所有 68 個車站
- 格式: "中文名 (English Name)"
- 選擇後立即更新

## 信息顯示

每個列車卡片包含：
1. **路線號碼** - 大字體、彩色顯示
2. **目的地** - 中文名稱，帶 → 箭頭
3. **到站時間** - ⏰ 圖示 + 時間（如 "2 min"）
4. **車廂類型** - 🚋 圖示 + 類型（單卡/雙卡）

## 性能優化

- CSS Grid 佈局（高效）
- CSS 動畫（GPU 加速）
- 最多顯示每個月台前 5 班列車
- 30 秒自動刷新（可配置）
